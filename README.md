# ApplianceRepairPortal
A web-based appliance repair management system for booking, tracking, and managing home appliance service requests using Java, JSP, Servlets, and MySQL.
ApplianceRepairPortal is a web-based application designed to simplify the process of booking and managing home appliance repair services. The system connects customers with service providers through an easy-to-use interface, allowing users to register, submit repair requests, track service status, and manage appointments efficiently.

Features
User registration and secure login
Online appliance repair request submission
Appliance category selection (TV, Refrigerator, Washing Machine, AC, etc.)
Service request tracking and status updates
Appointment scheduling
Customer profile management
Admin dashboard for managing users and repair requests
Technician assignment and service management
Responsive and user-friendly interface
Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Java (JSP/Servlets)
Database: MySQL
Server: Apache Tomcat
IDE: NetBeans / Eclipse
Objectives
Simplify the appliance repair booking process.
Reduce manual paperwork.
Improve communication between customers and service providers.
Provide efficient service request tracking and management.
Future Enhancements
Online payment integration
Email and SMS notifications
Live technician tracking
Customer reviews and ratings
AI-powered issue diagnosis and chatbot support
Mobile application support
Project Outcome

The ApplianceRepairPortal provides a centralized platform for managing appliance repair services, improving efficiency for both customers and administrators while delivering a seamless service experience.
